=====================
 Makefile for Golang
=====================

Requirements
------------

* Golang >= 1.2
* GNU make
  
Installation
------------
::
   
   $ git clone https://github.com/mkouhei/makefile-go.git
   $ install -d /path/to/somerepo
   $ cp makefile-go/Makefile /path/to/somerepo
   $ cd /path/to/somerepo
   $ make REPO=git@example.org/username/reponame.git
